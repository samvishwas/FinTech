# FinTech

A centralized hub for high-performance financial tools designed to bridge the gap between economic theory and personal strategy. 

## 🏗 Project Mission
To empower users with engineering-driven clarity, transforming "what-if" scenarios into data-backed decisions. This suite includes tools for mortgage optimization, inflation modeling, and (upcoming) derivative trading analysis.

## 📂 Current Tools

### /Calculators
* **Dhurandhar's Dilemma (`payoff-vs-invest.html`)**: A flagship single-file application to evaluate mortgage prepayment versus market investment.
    * **Live Data**: Real-time BTC and CPI inflation fetching.
    * **Real vs. Nominal**: Models how inflation erodes debt cost and impacts real purchasing power.
    * **Benchmarking**: Instant comparison against S&P 500, Nasdaq-100, and TSLA returns.

### /Trading (Coming Soon)
* **OptionsSphere**: Advanced portfolio and options tracking engine.
